虎扑 | 产品分析报告

2022-12-04

  虎扑的是一个社区类APP，从最初的一个体育论坛逐渐转变成一个多元内容社区。和知乎、豆瓣不同，虎扑依靠体育新闻资讯聚集大量JRs，通过创建不同主队社区板块，构建众多子社区，这些子社区才是虎扑的核心。本文将从产品概述、产品功能、用户画像等角度去分析虎扑社区。

产品概述

  • 体验环境
    设备：红米K30S
    操作系统：MIUI 13.0.6
    体验版本：v8.0.29
    体验网络：4G、无线网
    体验时间：2022.11.27 - 2022.12.1

  • 产品介绍

  1. 2004年1月CEO程杭博士在芝加哥创建HoopCHINA篮球论坛，Hoop的中文意思是篮圈、篮框。虎扑最开始是以篮球为主要方向，是一个典型的专注于篮球赛事的垂直社区。
  2. 2007年09月，成立公司运营，11月创建足球、F1、网球和C2C频道，开始进行多元化体育发展。
  3. 2012年05月，虎扑体育Android、iPhone手机客户端正式上线，进军移动互联网领域，同年10月份，赞助WCBA联赛，成为职业联赛合作伙伴，为其提供官网服务。
  4. 2014年，虎扑开始开辟购物板块，创办了导购网站“识货”，进军电商产业，到2017年交易规模达到了20亿，占据了虎扑商业化板块很大比重。
  5. 2019年6月份，字节跳动以12.6亿元人民币的价格收购了30%的股权，成为了虎扑第一大股东。
  6. 截至目前，虎扑包括篮球、足球、步行街、影视、电竞五大板块300多个话题区，从最初的一个体育论坛逐渐转变成一个年轻人的多元内容社区。
  • 产品定位
  1. 最开始的定位是专注于篮球赛事的垂直社区，为篮球爱好者提供篮球资讯和交流的平台。
  2. 随着不同体育板块的开通，虎扑的定位是多元体育平台，为各类体育爱好者提供交流的平台。
  3. 紧接着数码、影视等非体育板块的开通，越来越多年轻人加入虎扑社区，虎扑逐渐变成多元内容社区
  4. 现在，虎扑是一个为体育爱好者提供专业的体育资讯、赛事直播的体育平台，又是一个真实有趣有温度的年轻人社区。
  • 产品slogan
  1. ”虎扑，可能是最好的篮球网站“
  2. ”虎扑，你的体育全世界“
  3. ”世界是一个竞技场“
